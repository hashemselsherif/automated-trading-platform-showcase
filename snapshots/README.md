# Snapshots

Sanitized static outputs for quick project review. These files are representative review artifacts and do not include credentials, wallet material, private RPC endpoints, deployment IDs, raw databases, or full trade logs.

## Contents

- `dashboard/dashboard-snapshot.png`: static dashboard screenshot generated from sanitized state.
- `dashboard/dashboard-snapshot.html`: source HTML used to render the screenshot.
- `logs/startup-sanitized.log`: representative startup log excerpt with sensitive fields omitted.
- `logs/runtime-loop-sanitized.log`: representative runtime loop log excerpt with identifiers sanitized.
- `backtests/rsi-reversion-backtest-summary.json`: summarized RSI reversion backtest output.
- `backtests/rsi-reversion-backtest-summary.txt`: human-readable backtest summary.
